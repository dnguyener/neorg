get_node_text (node, buf)
Extracts the text from a node (only the first line)

Parameters:

  • node userdata a treesitter node to extract the text from
  • buf number the buffer number. This is required to verify the source of the node. Can be nil in which case it is treated as "0"

Returns:

    string The contents of the node in the form of a string
